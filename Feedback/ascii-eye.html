// About

This is a demo page showing the ASCII eye component in a matrix black-and-green CV theme. The eye lives in its own container (#eye-slot) and reacts to the mouse anywhere on the page — it only "wakes up" and tracks once the cursor comes within its proximity radius, and idles with a slow drifting motion otherwise.

// Notes

The eye is drawn on a single canvas. Characters aren't rerolled every frame — only a small random slice of cells changes each tick, which is what keeps the animation calm instead of flickering, and keeps it light on the CPU/GPU so it won't bog down the rest of the page.

To embed elsewhere: keep the <script> block below, give any element an id/ref, and call new AsciiEye(el). See the comments in the script for all options (color, size, proximity, speed).
